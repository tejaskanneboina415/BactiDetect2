Dataset Sources

This project uses publicly available, peer-reviewed datasets for training and evaluation. Below are the main sources:

DIBaS: Digital Images of Bacteria Species

Description: 33 bacteria species, 20 images per species. Stained using Gram's method, imaged at 100x oil-immersion.
Source: Chair of Microbiology, Jagiellonian University, Krakow, Poland.
Access: DIBaS Dataset Download
Citation: Zieliński B, Plichta A, Misztal K, Spurek P, Brzychczy-Włoch M, Ochońska D. Deep learning approach to bacterial colony classification. PLOS ONE, 12(9), 1-14, 2017. [Read Paper]
License: Open access, attribution required.

Clinical Bacterial Dataset for Deep Learning in Microbiological Rapid On-Site Evaluation

Description: 1,705 clinical Gram-stained images from respiratory specimens, with 4,833 cocci and 6,991 bacilli annotated.
Source: Chinese PLA General Hospital, Beijing, China.
Access: Zenodo Dataset
Citation: Wang, X., Shi, Y., Guo, S. et al. A Clinical Bacterial Dataset for Deep Learning in Microbiological Rapid On-Site Evaluation. Sci Data 11, 608 (2024). [Read Paper]
License: Creative Commons Attribution 4.0 International (CC BY 4.0)

Augmented Training Dataset

Description: Enhanced dataset created through data augmentation techniques to improve model accuracy. Expanded from 679 to 1,850 images across 32 bacterial species.
Source: Generated from original datasets using advanced augmentation techniques.
Augmentation Techniques: Rotation (±30°), brightness/contrast adjustment, noise addition, blur/sharpen, horizontal/vertical flips, and combinations thereof.
Target: 50 samples per class for balanced training.
Purpose: Address class imbalance and limited training data to improve model generalization.
Status: Generated and available for training in data/augmented/ directory.

Note: All datasets are used in accordance with their respective licenses. If you use this project or its data, please cite the original sources.
